# 概要
このプログラムは、ディレクトリ内の容量を調査するCLIソフトウェアです。　　
(Windows専用となっております)
# 導入方法
このプロジェクトを任意の場所に展開してください。
# 使い方
size-search.exeがあるディレクトリで以下のコマンドを実行してください。  
```size-search 任意のパス```  
すると、指定したパス内にあるすべてのファイルとディレクトリの容量がソートして出力されます。

# オプション
このコマンドには以下のオプションが用意されています。  
- `--per`: 各ファイルがどれほどの割合を占めるかを表示する
- `--file`: ファイルに絞り込んで表示する
- `--dir`: ディレクトリに絞り込んで表示する
- `--help`: ヘルプを表示する
# 注意
windows apiを使用しているため、windowsでのみ使用可能となります。ご了承ください。  
このプログラムは、C言語を用いて作成しております。
